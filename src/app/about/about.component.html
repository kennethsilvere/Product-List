About

A simple Angular 4 application made using TypeScript, can be used to store a list of Products or Items and their descriptions.

Some of the features of Angular 4 used in this app :

  • Components (Obviously)
  • Use of Directives
  • Services
  • Routes
  • Animations
  • Form Handling (Template Driven Approach)
  • String Interpolation, Property Binding, Two-Way data binding, Event Binding, etc
